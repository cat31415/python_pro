1 Задача:

Дан список целых чисел, представляющий температуры по дням за месяц (всего 30 значений).
Тебе нужно:

Вывести температуры только за первую декаду (первые 10 дней). создать отдельный список

Вывести температуры только за последнюю неделю (последние 7 дней). создать отдельный список

Вывести температуры через день (каждый второй элемент списка). создать отдельный список

Создать новый список, в котором температуры идут в обратном порядке.

temps = [3, 4, 2, 1, 0, -1, 1, 2, 5, 6,
         7, 8, 6, 5, 3, 2, 1, 0, -2, -3,
         -1, 1, 3, 5, 6, 7, 6, 4, 3, 2]

2 Задача: “Склад фруктов”

На складе хранится количество разных фруктов. Данные записаны в словаре:

warehouse = {
    "яблоки": 50,
    "бананы": 25,
    "груши": 18,
    "апельсины": 40
}


Тебе нужно:

1. Вывести весь список фруктов и их количество. (общее количество фруктов)

2. Пользователь вводит название фрукта.

Если фрукт есть на складе, вывести его количество.

Если нет — вывести сообщение: "Такого фрукта нет на складе".

3. Пользователь вводит название фрукта и число, которое хочет забрать.

Если на складе достаточно — уменьшить количество.

Если недостаточно — вывести: "Недостаточно запасов".

4. Добавить новый фрукт и его количество (оба вводятся пользователем).

Вывести обновлённый словарь.
