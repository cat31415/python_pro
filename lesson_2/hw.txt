Задания:

Выведи первые три точки пути.

Выведи все точки, начиная с "ручей" и до конца. используйте метод index() чтобы найти "ручей"

Создай список, в котором путь идет назад (от сундука к порту). 

Выведи каждую третью точку пути.

2) Списки — “Питомцы зоопарка”

animals = ["лев", "тигр", "слон", "медведь", "жираф"]


Задания:

Пользователь вводит имя нового животного — добавь его в список.

Если в списке есть "медведь", выведи: "Медведь спит в берлоге". использовать if .. in ... 

Удали животное, которое пользователь хочет отправить в другой зоопарк. remove() перед удалением проверить, что такое животное есть

Выведи количество животных.

Переверни список (чтобы порядок был обратным) и выведи.

3) Словари — “Рюкзак приключенца”

Словарь с предметами и их весом:

bag = {
    "веревка": 3,
    "зелье": 1,
    "камень": 5,
    "нож": 2
}


Задания:

Выведи каждый предмет и его вес.

Пользователь вводит название предмета — выведи его вес.
Если нет — напиши: "В рюкзаке такого нет".

Пользователь находит новый предмет — добавь его с весом.

Игрок выбрасывает самый тяжелый предмет — удали его. del bag["нож"]

Выведи общее суммарное количество веса рюкзака.
