Задание: «Подсчёт строк и слов в файле»
Цель

Научиться:

читать текстовый файл

использовать функции

считать строки и слова

Условие

Дан текстовый файл text.txt.

Нужно написать программу, которая:

Считает, сколько строк в файле

Считает, сколько слов в файле

Выводит результат на экран

Требования

Использовать две функции:

count_lines(filename) — считает количество строк возвращает через return 

count_words(filename) — считает количество слов
