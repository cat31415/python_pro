Цель: написать программу, которая читает текст из файла и выводит статистику, используя функции.

Входные данные

Текстовый файл text.txt (несколько строк).

Что нужно сделать

Написать функцию read_text(filename), которая читает файл и возвращает весь текст одной строкой.

Написать функцию clean_word(word), которая: приводит слово к нижнему регистру

clean_text(text): убирает знаки препинания по краям (например .,!?":;())

Написать функцию count_stats(text), которая возвращает:

количество строк

количество слов

количество уникальных слов 

самое частое слово и сколько раз оно встречается

Написать функцию save_report(filename, stats), которая сохраняет результат в файл report.txt.
